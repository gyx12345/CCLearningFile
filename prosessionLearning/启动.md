# 启动 

## 阶段0：

rom里检查电压，外设

从emmc或nanda flash 加载uboot到SRam

## 阶段1：

uboot 初始化 DRam（内存）

加载linux kernel。

加载dtb。通过bootargs传递启动参数给linux kernel



## 阶段2：

linux kernel 解压缩

initramfs 解压到内存（cpio.gz）作为临时/  --->执行initramfs里的 /init  (**存在内存里的驱动**) （**解决需要读磁盘驱动拿到文件，但读磁盘需要驱动**)

rootfs 初始根文件系统 里面至少要 1、/sbin/init 或 /init  2、/bin /sbin 或 /lib  3、/etc   4、/dev 或 /proc 或 /sys

初始化init线程。
